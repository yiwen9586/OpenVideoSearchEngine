# OpenVideoSearchEngine
INLS 760 Web Database Project2

Implemented full-text search capabilities for data stored in a MySQL databse which extracted from OpenVideo web site. Used session contol to implement user login and logout. Used jQuery+Ajax to implement a “search suggestion” interface and a mouse-over feature to show document details.

## Login -- Session control
You can use username: rob, password: rob to login to the main page
![login-page.png](https://github.com/yiwen9586/OpenVideoSearchEngine/blob/master/screenshots/login-page.png)

## Main Page -- Keyword Suggestion
When users type queries into the input box, the suggestion div will show key word suggestions based on data stored in the database.
![type-suggestion.png](https://github.com/yiwen9586/OpenVideoSearchEngine/blob/master/screenshots/type-suggestion.png)

## Main Page -- Search Results
After user clicking the search button, full text search results would be displayed in the page.
![search-results.png](https://github.com/yiwen9586/OpenVideoSearchEngine/blob/master/screenshots/search-results.png)

## Main Page -- Show Details -- Ajax
When user hovers a row of the search results, the document details will be displayed on the right side.
![show-details.png](https://github.com/yiwen9586/OpenVideoSearchEngine/blob/master/screenshots/show-details.png)
